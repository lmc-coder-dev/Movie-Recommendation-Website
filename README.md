## 🎬 电影推荐网站

### 📁 此仓库相关介绍

本仓库存放纯前端三件套打造的电影推荐网站项目，全程基于 HTML、CSS、JavaScript 开发，未对接后端接口数据

页面跳转均外链至第三方站点，这是我入门前端时，参考思路一步步实操完成的作品😆

如今回看页面样式略显朴素，但当初完成项目时成就感满满，对我而言有着特别的成长意义✨
